# online-dating-app

## Description
オンラインで相手を探してメッセージのやり取りができるSPAアプリです。  

ルーティング管理にVue-Router、状態管理にVuexを使用しました。

また以下を使用しています。

* RANDOM USER GENERATOR  
    ランダムなユーザー情報を取得できるAPI

* Talk API  
    入力テキストから応答文を生成することで日常会話を可能にするAPI

* Leaflet  
    Web地図のためのJavaScriptライブラリ
